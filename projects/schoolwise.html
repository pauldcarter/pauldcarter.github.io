Schoolwise

League tables tell you who had a good year. Schoolwise shows which UK schools are strong over many years — trajectories, not leaderboards — searchable by postcode on an interactive map.

In development

Multi-year trajectories, not leaderboards

A single-year ranking rewards a lucky cohort. Plotting each school's results across years shows consistency — and a byte-for-byte raw audit layer in the database means every displayed number can be traced back to the exact government file it came from.

Key features

  • Multi-year performance trajectories per school, instead of single-year league-table snapshots.
  • Interactive map search — enter a postcode, see the schools around it with their long-run record.
  • Byte-for-byte raw audit layer: every published figure is traceable to the original source file.
  • Portable Python ingest pipeline that reprocesses each yearly government data release.
  • Open data with no sign-in — the whole app is readable by anyone.

How it's built

Front end

Next.js (App Router) + TypeScript + Tailwind SSR + typed components for a data-heavy UI.

Back end

Supabase Edge Functions + Python pipeline Auto REST over the data; the Python ingest stays portable.

APIs / Auth

PostgREST · MapLibre/Mapbox · postcodes.io Generated API off Postgres; interactive maps with postcode geocoding; no user auth needed.

Hosting

Vercel + Supabase One thing to deploy, nothing to operate.

Database

Postgres + PostGIS Geo queries and auto-generated REST out of the box.
