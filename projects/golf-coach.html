Golf Coach

A golf coaching app built the safe way round: a coach curates a visual knowledge base of technique — foundations, symptoms and fixes — and the AI is only allowed to recommend from that approved library.

In development

GUI-managed knowledge driving the coach

Most AI coaches make things up. Here an admin edits a mind-map graph of foundations → symptoms → resources, and the logic/AI coach diagnoses a player's technique and prescribes only fixes from that curated library — the human stays in charge of the advice.

Key features

  • Visual mind-map editor for the knowledge base — drag, connect and auto-layout, hand-built in SVG/canvas.
  • A structured graph of foundations → symptoms → resources that encodes real coaching knowledge.
  • A logic/AI coach that diagnoses technique from symptoms and prescribes only approved fixes.
  • Zero dependencies front to back — vanilla JS in the browser, plain Node http on the server.
  • Atomic write-and-rename saves to a JSON file store — safe persistence without a database.

How it's built

Front end

Vanilla JS · SVG/canvas mind-map Hand-built drag/connect/auto-layout graph editor, zero front-end dependencies.

Back end

Zero-dependency Node server Plain Node http — no Express; serves the editor and the knowledge API.

APIs / Auth

Internal save/export API Reads/writes the knowledge base; AI recommends only from the curated library.

Hosting

Local / self-host Runs on Node anywhere; not yet publicly deployed.

Database

JSON file store Atomic write-and-rename to knowledge-base.json; right-sized for a curated graph.
