Family Manager

An app for organising family life that installs to the home screen like a native app and keeps working with no signal — because there's no server, no account and no network dependency at all.

Visit live ↗

Installs like a native app, works offline

The interesting decision here is what was left out. No back end, no database host, no sign-in — the PWA app shell and on-device storage do everything, which means zero running costs, total privacy and an app that works on a plane.

Key features

  • Add-to-home-screen PWA — installs like a native app on phones and desktops.
  • Fully offline-capable: the app shell and data live on the device.
  • No account and no sign-in — open it and start using it.
  • Complete privacy by architecture: family data never leaves the device.
  • Zero running costs — static hosting is the entire infrastructure.

How it's built

Front end

React 18 + Vite + React Router (PWA) Installable, offline-capable app shell.

Back end

None — client-side All logic in the browser; nothing to run server-side.

APIs / Auth

None Single-user on-device app; no API or sign-in needed.

Hosting

Vercel Zero-config SPA hosting with instant deploys.

Database

Client-side (PWA storage) Data stays on-device; works without a network.
